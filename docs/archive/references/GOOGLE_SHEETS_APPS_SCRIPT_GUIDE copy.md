# GOOGLE_SHEETS_APPS_SCRIPT_GUIDE.md (Quick Install)

1) Open your Sheet → Extensions → Apps Script.
2) Replace contents with `apps_script/google_sheets_dashboard_v2.gs`.
3) Save, then run `Setup_Dashboard_AutoRefresh()` and authorize.
4) Use menu **⚡ Power Market** to refresh/rebuild/enable auto-refresh.
